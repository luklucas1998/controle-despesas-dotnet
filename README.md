# 💻 Sistema de Controle de Despesas

Aplicação web desenvolvida com ASP.NET Core MVC (.NET 8)

## 🚀 Funcionalidades
- Cadastro de despesas
- Listagem de dados
- Integração com banco SQLite

## 🛠 Tecnologias
- C#
- ASP.NET Core
- Entity Framework
- SQLite

## ▶️ Como executar
1. Clone o repositório
2. Rode Update-Database
3. Execute o projeto
